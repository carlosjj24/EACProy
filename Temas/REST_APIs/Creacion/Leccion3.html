API REST de Twitter

       Acontinuacioón haremos un estudio de las API REST de Twitter; en su mayoria las API REST mas usadas en la actualidad pertenecen a las tecnologias emergentes con la era 2.0.



API REST de Twitter

       A partir de la versión 1.1, la API de Twitter ahora requiere autenticación OAuth, ya sea autenticación de aplicación única o autenticación de usuario de aplicación.

       Twitter presenta una serie de API segun su funcionalidad, las mas interesantes son:

  • Buscar Tweets.

  • Filtrar Tweets en tiempo real.

  • Twitter para sitios web.

  • API de mensaje directo.



¿Qué nos ofrece el API REST de Twitter?



Buscar Tweets

    La API de búsqueda estándar de Twitter (búsqueda / tweets) permite realizar consultas sencillas contra los índices de los Tweets recientes o populares y se comporta de manera similar, pero no exactamente como la función de UI de búsqueda disponible en los clientes móviles o web de Twitter. El buscador de Twitter busca en una muestra de los últimos tweets publicados en los últimos 7 días.

    Twitter proporciona una UI para probrar el API



Solicitud al API de Twitter



Respuesta del API de Twitter

    Esta primera parte del la respuesta del API de Twitter corresponde los datos básicos que puede poseer un tweet publicado.



Respuesta del API de Twitter

    Esta segunda parte del la respuesta del API de Twitter corresponde al encabezado de la metada del tweet corresponde a los datos del creador (id, nickname, descripción), en que lenguaje esta escrito el tweet (en-Ingles, es-Español,…), url directo del tweet, entre otros.



Respuesta del API de Twitter

    Por ultimo esta parte del la respuesta del API de Twitter corresponde otra parte de la metadata la cual hace referencia a información mas detallada, entre los que se encuentra: un booleano que representa la protección del tweet, lenguaje del perfil del creador, imagen de perfil del creador del tweet, colores del perfil del creador, entre otros.
