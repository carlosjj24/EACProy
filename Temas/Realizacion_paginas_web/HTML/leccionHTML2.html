HTML - Lección 2.1

Etiquetas Básicas - Parte 1

       A continuación se introducirá a las etiquetas básicas de HTML a traves de las cuales se podrá realizar una página web. En la primera parte se explicarán únicamente las etiquetas a utilizar entre la etiqueta <head>.



<link>

       La presente etiqueta permite el enlace de archivos externos al documento HTML actual. Por los momentos se utilizará para agregar una imagen a la pestaña del navegador. Para realizar la acción mencionada la etiqueta debe ser utilizada de la siguiente manera:

<link rel="icon" href="ruta imagen" type="image/extensión" size="16x16">


* rel: indica la relación del enlace, en nuestro caso es un icono, de ahí el valor del campo.

* href: indica la localización del documento a enlazar.

* type: como su nombre lo indica, define el tipo de documento a enlazar.

* size: propiedad valida unicamente para 'rel="icon"', e indica el tamaño del mismo.



<link> - Ejemplo


<meta>

       La etiqueta <meta> permite definir metadatos de la página. Actualmente lo utilizaremos para definir la codificación de caracteres a utilizar, con la finalidad de poder mostrar los acentos correctamente. Ejemplo:

<meta charset="UTF-8">



<title>

       A traves de la etiqueta <title> es posible definir el título de la página HTML. La implementación de la etiqueta mencionada se realiza de la siguiente manera:

<title> Título de la Página </title>
