HTML - Leccion 2.2

Etiquetas Básicas - Parte 2

       A continuación se introducirá a las etiquetas básicas de HTML a traves de las cuales se podrá realizar una página web. En la segunda parte se explicarán las etiquetas utilizadas entre la etiqueta <body>.



<img>

       La etiqueta img permite agregar una imagen en el cuerpo de la página web. Para realizar la acción mencionada la etiqueta debe ser utilizada de la siguiente manera:

<img src="ubicación imagen" alt="descripción imagen">


* src: indica la localización de la imagen.

* alt: especifica un texto alternativo a la imagen.

Etiquetas: <p>, <b>, <i>

       Son las etiquetas más básicas en HTML. la etiqueta p indica un nuevo parrafo, mientras que las etiquetas i y b indican letra cursiva o negrita respectivamente. Las tres etiquetas son utilizadas de la misma forma:

<p> Nuevo parrafo </p>



Etiquetas: <p>, <b>, <i> - Ejemplo


Tablas

       Con HTML se pueden utilizar tablas que organicen la información de una mejor manera según sea el caso. A continuación se muestra un ejemplo para la creación de una tabla sencilla, donde <tr> define las filas y <td> las columnas:

<table>
     <tr>
         <td> <!-- Primera fila, primera columna. --> </td>
         <td> <!-- Primera fila, segunda columna. --> </td>
     </tr>
     <tr>
         <td> <!-- Segunda fila, primera columna. --> </td>
         <td> <!-- Segunda fila, segunda columna. --> </td>
     </tr>
</table>



Tablas - Ejemplo


<div>

       Con la etiqueta <div> puede agruparse el contenido de la página, con la finalidad de cambiar sus estilos por separado, o simplemente tener la información mas organizada. Ejemplo:

<div>
     <!-- Contenido de una sección de la página. -->
</div>
