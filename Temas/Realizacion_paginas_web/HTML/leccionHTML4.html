HTML - Lección 3

Formularios

       En la presente lección explicaremos ccmo obtener información de los usuarios a traves de los formularios HTML.



<form>

       La etiqueta <form> es la encargada de indicar el inicio, y el fin, de un formulario HTML. Ejemplo:

<form>
     <!-- Contenido del formulario. -->
</form>



<input>

       La etiqueta input indica un campo de entrada a traves del cual el usuario puede introducir datos.

<input type="tipo" name="nombre" value="valor">


* type: como su nombre lo indica, define el tipo de input. Segun su valor puede cambiar el comportamiento del elemento HTML.

* name: indica el nombre del elemento HTML.

* value: indica el valor del campo.



<input>: Tipos



<select>

       La etiqueta <select> es utilizada para la creación de una lista desplegable. Ejemplo:

<select>

    <option value="valor"> <!-- Nombre de la opción --> </option>

</select>




<textarea>

       El textarea define una entrada de texto de multiples lineas. Su funcionamiento es basicamente igual al de un input de tipo text. Ejemplo de implementación:

<textarea>

    <!-- Valor por Defecto -->

</textarea>
