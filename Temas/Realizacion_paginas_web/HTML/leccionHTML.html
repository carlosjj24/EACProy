HTML - Lección 1

¿Que es?

       El Lenguaje de Marcado para Hipertextos, o HyperText Markup Language (HTML) en ingles, es una herramienta utilizada para el desarrollo de páginas web. Dicho lenguaje se encuentra compuesto por una serie de etiquetas las cuales son interpretadas por el navegador, dando forma a la pantalla.



Etiquetas

       Las etiquetas HTML son fragmentos de texto rodeados por corchetes angulares < >, que tienen funciones y usos específicos y se utilizan para escribir código HTML. Las etiquetas puden o no poseer una etiqueta de cierre.



¿Cómo iniciar?

       Comenzar la creación de una página HTML es un procedimiento muy sencillo, lo único que se debe hacer es:

1- Dirigirse al directorio donde se desea crear el proyecto

2- Dar click derecho dentro del directorio

3- Seleccionar "nuevo" y luego "documento de texto"

4- Seleccionar el nombre completo del archivo, incluyendo la extensión del mismo, y colocar nombre.html

5- Abrir con el editor de texto de su preferencia

6- Listo! ya puedes comenzar a crear tu página web



Estructura Básica

       Los documentos HTML poseen una estructura básica a traves de la cual se puede relizar el desarrollo de la página.

<!DOCTYPE html>

<html>

    <head>

        <!-- metadatos de la página -->

    </head>

    <body>

        <!-- contenido visible de la página -->

    </body>

</html>
