CSS - Lección 3

Modificación de estilo en general

       A continuación te enseñaremos lo necesario para modificar el estilo de la página web en general.



Propiedades

       Entre algunas de las propiedades que permiten el cambio de estilo de los elementos de la página web se encuentra:

* background-color: indica el color del fondo.

* background-image: indica la imagen de fondo

* width: indica el ancho del elemento.

* heigth: indica el alto del elemento.

* padding: es utilizado para generar espacio al rededor de un objeto contenido en cualquier borde definido.

* margin: es utilizado para generar espacio al rededor de un objeto fuera de cualquier borde definido.
