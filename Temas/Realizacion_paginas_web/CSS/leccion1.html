CSS - Lección 1

¿Que es?

       Las Hojas de Estilo en Cascada o CSS, por sus siglas en ingles de Cascading Style Sheets, es un lenguaje de diseño utilizado para dar estilo y distribuir el contenido de los documentos estructurados escritos en lenguaje de marcado.



Clases y IDs en documentos HTML

       La etiquetas HTML poseen atributos que puden ser utilizados, ejemplos de estos atributos son el src de la etiqueta <img> o el href de la etiqueta <link>. Adicionales a los atributos ya mencionados existen otros dos que pueden ser utilizados en cualquier etiqueta, estos son los IDs(id) y las clases(class); los primeros sirver para identificar una etiqueta única, mientras que los segundos sirven para identificar multiples etiquetas simultaneamente. Esto es importante tenerlo en cuenta debido a que más adelanta servirán de ayuda para la definición de los estilos.



¿Cómo iniciar?

       La realización del documento CSS es igual que en el caso de HTML, la única diferencia es que la extensión sera ".css" en lugar de ".html".



Estructura Básica

       Los documentos CSS tienen una estructura muy sencilla:

selector{

     /*Propiedad: valor;*/

}



Selectores

       Los selectores sirven para declarar a cuales etiquetas se les aplicaran los estilos deseados. Existen muchos selectores, sin embargo los más básicos son:

* Etiqueta

* .Clase

* #ID



Enlazar CSS a documento HTML

       Luego de haber creado el documento HTML y CSS es momento de enlazarlos, esto se hace a traves del documento HTML, agregando entre la etiqueta <head> la siguiente etiqueta:

<link rel="stylesheet" href="ubicación del archivo CSS" type="text/css">
