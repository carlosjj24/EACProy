CSS - Lección 3

Unidades

       CSS permite colocar los valores de las propiedades de muchas maneras, en especial los colores y tamaños. A continuación se esplican algunas unidades posibles en un archivo CSS.



Unidades de color

       En CSS se pueden escribir los colores de tres maneras distintas:

* RGB: para indicar el color en formato RGB el valor debe ser definido de la siguiente manera: rgb(r,g,b) donde r,g,b indica la cantidad que posee de cada uno de los colores.

* hexadecimal: para indicar el color en formato hexadecimal debe estar definido de la siguiente manera: #numero hexadecimal.

* nombre del color: algunos color se pueden especificar por sus nombres en ingles, entre estos esta: blue, green, orange, black, white, etc.



Unidades de medidas absolutas

       Entre las unidades de medidas absolutas existentes en CSS existen:

* cm: centimetros.

* mm: milimetros.

* in: pulgadas.

* pt: puntos.

* pt: picas.



Unidades de medidas relativas

       Entre las unidades de medidas relativas existentes se encuentran:

* em: correspondiente al tamaño actual de la letra.

* ch: relativo al tamaño del 0.

* rem: correspondiente al tamaño de letra del elemento raiz.

* vw: relativo al 1% del ancho de la pantalla.

* vh: relativo al 1% del alto de la pantalla.

* %: relativo al elemento padre.
