JavaScript - Lección 2

Variables

       En la presente lección se mostraran los tipos de variables existentes en JavaScript



Variables generales

       En JavaScript no es necesario especificar el tipo de variable, por lo cual solo es necesario asignarle un valor para definir su tipo. Ejemplo:

var a="Hola";

var b=20;

var c='a';



Arreglos

       JavaScript permite guardar multiples valores en una simple variable, esto se logra a traves de los arreglos. Ejemplo:

var array=["Primero", "Segundo", "Tercero"];

//Acceso a un elemento de la variable

var primero=array[0];

var segundo=array[1];

var tercero=array[2];



JSON

       Un JSON, o JavaScript Object Notation, es, de forma coloquial, una especie de arreglo el cual puede ser indexado por una clave en lugar de un numero. Ejemplo:

var json={"Primero":"valor1", "Segundo":"valor2", "Tercero":"valor3"};

//Acceso a un elemento de la variable

var primero=json.primero;

var segundo=json.segundo;

var tercero=json.tercero;
