JavaScript - Lección 3

Eventos

       Una de las funcionalidades más útiles de JavaScript son los eventos, estos permiten ejecutar una función dada una acción dentro de la pagina.



Tipos de Eventos

       Exiten multiples eventos que pueden ayudar a hacer la página web dinamica, entre estos se encuentran:

* onchange: se activa cuando ocurre un cambio en el elemento HTML.

* onclick: se activa cuando se realiza un click sobre el elemento HTML.

* onmouseover: se activa cuando el usuario posiciona el mouse sobre el elemento HTML.

* onmouseout: se activa cuando el usuario deja de posicionar el mouse sobre el elemento HTML.

* onkeydown: se activa cuando el usuario presiona alguna tecla del teclado.

* onload: se activa cuando el navegador termina de cargar la página.



Funciones

       Una función de JavaScript es un fragmento de codigo que puede ser invocado para su posterior ejecucion. Ejemplo:

function nombre(Parametro 1, Parametro 2, ...Parametro n) {

    //Codigo JS

}
